Key Features

🚀

Multi-Bot Orchestration

Run multiple bots from a single process with shared context, centralized config, and independent conversation sessions.

⚙️

Autonomous Agent Loop

Self-directed execution loop with planning, tool use, and reflection cycles that operate without user prompts.

🧠

4-Layer Memory

Session history, daily memory logs, long-term knowledge base, and soul/personality persistence across restarts.

🔧

LLM Tool System

27 tools registered dynamically with JSON-schema parameters, collaboration-safe filtering, and sandboxed execution.

🤝

Multi-Bot Collaboration

Bot-to-bot delegation, visible and internal messaging, multi-turn conversation chains, and deference protocols.

📊

Web Dashboard

Real-time monitoring UI with conversation logs, memory viewer, agent loop status, and production management.

👻

Soul System

Persistent personality layer with evolving traits, mood tracking, and introspective self-modification capabilities.

🧩

Skill Framework

Modular skill plugins with skill.json manifests, isolated handlers, and hot-reload support.

⏰

Cron Scheduling

Scheduled tasks for memory consolidation, proactive messaging, soul reflection, and automated maintenance routines.
